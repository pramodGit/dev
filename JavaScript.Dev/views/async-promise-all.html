Async/await can introduce unnecessary blocking and sequential execution:

    const asyncFunc = async (test) => {
        console.log(new Date().getTime());
        const details = await test.map(item => item + ' 10');
        console.log(details);
        console.log(new Date().getTime());
      }
      asyncFunc(['Pramod', 'Kumar']);

Run Async Functions concurrently Using Promise.all:

    const promiseFunc = async (test) => {
        console.log(new Date().getTime());
        const details = await Promise.all(
          test.map(item => item + ' 12')  
        );
        console.log(details);
        console.log(new Date().getTime());
        return details;
      }
      promiseFunc(['pramod', 'Kumar']);
