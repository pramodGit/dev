Async Functions concurrently Using Promise.all & Promise.allSettled

Created three promises with success response

    const fetchUser = new Promise((resolve) => {
        setTimeout(() => resolve({ id: 1, name: 'Alice' }), 1000);
    });

    function fetchOrders() {
      return new Promise((resolve) => {
        setTimeout(() => resolve([{ id: 101, item: 'Book' }]), 1500);
      });
    }

    function fetchCart() {
      return new Promise((resolve, reject) => {
        setTimeout(() => resolve({ id: 120, item: 'Pen' }), 1200);
      });
    }

Now Run all simulated endpoints concurrently with Promise.all

    Promise.all([fetchUser, fetchOrders(), fetchCart()])
      .then(([user, orders, cart]) => {
        let combinedResult = { user, orders, cart };
        console.log('Combined Result:', combinedResult);
      })
      .catch((error) => {
        console.error('At least one request failed:', error.message);
      });

All promises has success response as shown below:

  Combined Result: {
    user: { id: 1, name: 'Alice' },
    orders: [ { id: 101, item: 'Book' } ],
    cart: { id: 120, item: 'Pen' }
  }

If set one Promise to send error response as shown below:

  function fetchCart() {
    return new Promise((resolve, reject) => {
      setTimeout(() => reject(new Error('Cart service failed')), 1200);
    });
  }

Output of the same promise.all:

  At least one request failed: Cart service failed

But allSettled provides better response object which is easy to handle in both cases

  Promise.allSettled([fetchUser, fetchOrders(), fetchCart()])
      .then(([user, orders, cart]) => {
        let combinedResult = { user, orders, cart };
        console.log('Combined Result:', combinedResult);
      })
      .catch((error) => {
        console.error('At least one request failed:', error.message);
      });

With error output of 'Promise.allSettled':

    Combined Result: {
      user: { status: 'fulfilled', value: { id: 1, name: 'Alice' } },
      orders: { status: 'fulfilled', value: [ [Object] ] },
      cart: {
        status: 'rejected',
        reason: Error: Cart service failed
            at Timeout._onTimeout (/usr/src/app/main.js:12:29)
            at listOnTimeout (node:internal/timers:588:17)
            at process.processTimers (node:internal/timers:523:7)
      }
    }

If you correct the error response then output of 'Promise.allSettled' will be:

    Combined Result: {
      user: { status: 'fulfilled', value: { id: 1, name: 'Alice' } },
      orders: { status: 'fulfilled', value: [ [Object] ] },
      cart: { status: 'fulfilled', value: { id: 120, item: 'Pen' } }
    }
